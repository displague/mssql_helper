# MSSQL Helper

PDO_SQLSRV wrapper for legacy MSSQL PHP database functions

## Usage

Include this file and legacy code that relies on ``sqlsrv_*`` database calls will work in a modern environment utilizing PDO\_SQLSRV drivers.
